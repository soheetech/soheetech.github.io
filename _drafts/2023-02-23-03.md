제어문(control)


조건문: 프로그램 수행 흐름을 바꾸는 역하을 하는 제어문 중 하나로 조건에 따라 다른 문장이 수행되도록 함

if문: 만약 ~라면
```
if(조건식1) {
  수행될 코드;
} else if(조건식2) {
  수행될 코드;
} else {
  수행될 코드;
}
```

switch문
```
switch(조건식) {

}
```
