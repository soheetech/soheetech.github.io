# instanceof 연산자
- 현재 참조형 변수가 어떤 클래스형의 객체 주소를 참조하고 있는지 확인할 때 사용
- 클래스 타입이 맞으면 `true`, 맞지 않으면 `false` 반환

## 표현식

```java
if(레퍼런스 instanceof 클래스타입) {
	// true일 때 처리할 내용
}
```
